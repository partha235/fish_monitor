# Fish Monitor 
The fish monitor kit for finding presence of fish with esp32-cam and OpenCV. 

### Hardware 
1) Esp32 S3 Dev kit -Cam  = Rs. 1490
2) GPS Sensor
3) Temperature Probe
4) DHT11 / BPM180
5) Waterproof Ultrasonic Sonic sensor
6) Water Quality sensor.
7) Display
8) esp32 dev-kit


## Abstract 

This project presents a smart fish detection and water monitoring system using an ESP32-S3 CAM with an OV3660 camera. The system applies embedded computer vision to detect fish and monitor their activity in real time. It integrates sensors such as a DS18B20 temperature probe, ultrasonic water level sensor, GPS module, and water quality sensor to measure environmental conditions. The collected data is processed on the ESP32-S3 and displayed locally or through a web interface. The system enables continuous monitoring of fish presence and water parameters. It provides a cost-effective and intelligent solution for modern aquaculture and environmental management.
